# ReLU and Its Improvements

This project demonstrates ReLU, its limitations (dying ReLU), and improvements like Leaky ReLU, PReLU, ELU, and Swish using Python.

## Files
- relu_analysis.ipynb
- ReLU_Improvements_report.pdf

## How to Run
pip install numpy matplotlib  
python relu_analysis.py
